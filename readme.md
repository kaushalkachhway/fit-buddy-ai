# FitBuddy AI – Fitness Plan Generator

FitBuddy AI is an intelligent fitness assistant that generates personalized workout plans using AI.

## Features
- AI-generated workout plans
- User fitness goal analysis
- Beginner to advanced workouts
- Clean web interface

## Tech Stack
- Java (Spring Boot)
- HTML, CSS, JavaScript
- Google Gemini AI API

## Project Structure
- Backend: Java API
- Frontend: Web interface
- AI Integration: Gemini API

## How to Run
1. Clone the repository
2. Install dependencies
3. Run the application
